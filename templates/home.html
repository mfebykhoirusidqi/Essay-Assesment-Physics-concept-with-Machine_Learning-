ABOUT THE SYSTEM

APA ITU GRADE AI SYSTEM

Grade AI System adalah sebuah web yang berfungsi sebagai asisten guru dalam mengoreksi atau penilaian jawaban siswa. Web ini dibuat menggunakan system NLP dan mechine learning pada Python serta penggunaan framework Flask sebagai penopangnya. Web ini berjalan secara otomatis dalam menilai hasil ujian/tes siswa dalam bentuk CBT essai.....

VISI

Grade AI System menjadi bukti dari kemajuan teknologi pendidikan dibidang penilaian. Selain itu, perkembangan teknologi informasi dan komunikasi telah membuka pintu bagi kemajuan dalam penilaian berbasis digital. Penilaian digital memanfaatkan teknologi untuk mengumpulkan, menyimpan, dan menganalisis data penilaian dengan lebih efisien. Ini juga memberikan fleksibilitas dalam metode penilaian, seperti penggunaan tugas proyek, portofolio elektronik, atau penilaian berbasis Ai.

FLASK FRAMEWORK

Flask adalah kerangka kerja aplikasi web mikro yang ditulis dalam bahasa Python. Flask bersifat ringan dan mudah digunakan, sehingga cocok untuk pemula. Flask juga memiliki komunitas pengguna yang besar dan aktif, sehingga tersedia banyak sumber daya untuk belajar dan pengembangan..

MACHINE LEARNING

Machine learning adalah cabang dari kecerdasan buatan yang berfokus pada pengembangan algoritma dan model statistik yang memungkinkan komputer untuk belajar dari data dan pengalaman, serta melakukan prediksi dan pengambilan keputusan tanpa harus secara eksplisit diprogram. Penerapan machine learning pada essay otomatis tidak hanya membantu menghemat waktu dan usaha dalam menulis, tetapi juga dapat memperluas aksesibilitas terhadap pendidikan dan literasi..
